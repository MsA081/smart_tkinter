# Tkinter GUI Builder

این پروژه یک ابزار گرافیکی برای طراحی رابط‌های Tkinter در پایتون است. با استفاده از آن می‌توانید ویجت‌ها را اضافه، جابجا، و تنظیم کنید و در نهایت طراحی خود را به صورت کد خروجی بگیرید.

## ویژگی‌ها

- اضافه کردن ویجت‌های مختلف (Button, Label, Entry, Text, ...)
- قابلیت درگ کردن ویجت‌ها
- پنل تنظیمات برای تغییر ویژگی‌ها
- نوار وضعیت برای نمایش پیام‌ها
- پشتیبانی از چند صفحه (Frame)
- خروجی گرفتن به صورت فایل `generated_gui.py`
- ذخیره‌سازی طراحی در `assets/config.json`

## اجرا

```bash
python builder.py
